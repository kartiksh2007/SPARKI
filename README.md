## 🤖 About the Project
**Sparki AI** is a JARVIS-inspired multimodal assistant developed by **Kartik (Sparkedix)**. It combines the power of NVIDIA's high-speed inference with a sleek, dark-themed UI to provide a seamless AI experience.

### 🔥 Key Features
- **NVIDIA Gemma Engine:** Super-fast streaming responses with low latency.
- **Flux.1-Dev Integration:** High-quality image generation using Black Forest Labs via NVIDIA.
- **Voice Integration:** Real-time ASR (Speech-to-Text) and TTS (Text-to-Speech).
- **Web Automation:** Smart intent recognition to open YouTube, Google, and more.
- **System Monitoring:** Battery and performance tracking directly in the UI.
